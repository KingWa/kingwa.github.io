Share Code Snippets

Apr 29th, 2014

Ways to share code snippets in Octopress:

Use the pygment’s highlight

The syntax is: 

{% highlight language linenos %}
{% endhighlight %}

where language is the pygment’s supporting language, linenos(optional) means showing line number.

For example, bellow:

{% highlight c %}
int main(int argc, char* argv[])
{
    printf("Hello, world!\n");
    return 0;
}
{% endhighlight %}

shows:

int main(int argc, char* argv[])
{
    printf("Hello, world!\n");
    return 0;
}

Simply use markdown

To produce a code block in Markdown, simply indent every line of the block by at least 4 spaces or 1 tab. Such as:

int main(int argc, char* argv[])
{
    printf("Hello, world!\n");
    return 0;
}

Backtick Code Blocks

Syntax:

``` [language] [title] [url] [link text] [linenos:false] [start:#] [mark:#,#-#]
code snippet
```

For example, bellow:

``` 
int main(int argc, char* argv[])
{
    printf("%s\n", "Hello, world!");
    return 0;
}
```

shows:

and 

``` c  
int main(int argc, char* argv[])
{
    printf("%s\n", "Hello, world!");
    return 0;
}
```

shows:

and

``` c "Hello World.cpp" mark:3 
int main(int argc, char* argv[])
{
    printf("%s\n", "Hello, world!");
    return 0;
}
```
